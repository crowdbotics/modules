# Youtube Player React native specs

## Module description

The Youtube Player Module is a React Native-based module which embeds a youtube video into your react native app.

- Change the quality of video.
- Pause the video.
- Fullscreen mode.
- Play video on loop.
- Previous and Next options.
- Share the video.
